# Job Board

Welcome to the Job Board project! This web application is built using React and styled with Tailwind CSS. It fetches the latest job postings from the Hacker News API and displays them on a user-friendly interface. ## Getting Started

1. **Clone the Repository**

   ```bash
   git clone https://github.com/sujitmemane/CureSkin_Assignment.git
   cd CureSkin_Assignmnent
   ```

2. **Install Dependencies**
   
           npm install
   
3. Start the Development Server

          npm run dev
